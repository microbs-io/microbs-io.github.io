Quick start

You can install microbs in three basic steps:

  1. Prepare your environment
  2. Clone the repo
  3. Install microbs

Once installed, you can proceed to the configuration and CLI.

Step 1. Prepare your environment

Operating system

microbs has been tested on MacOS 12.1. Your mileage may vary on other operating systems.

Software dependencies

Plugins may require additional dependencies. Be sure to read the docs for the plugins you wish to use.

You will need to install node, nvm, kubectl, and skaffold before installing microbs.

microbs has been tested with the following software versions:

Software Version
kubectl 1.23.4
node 17.5.0
nvm 17.5.0
skaffold 1.36.0

Step 2. Clone the repo

git clone https://github.com/microbs-io/microbs.git

Step 3. Install microbs

# Navigate into the cloned microbs repo
cd microbs

# Use the correct version of node
nvm use

# Install the node modules for microbs
npm install

# Add the current directory to your PATH
export PATH="$(pwd):$PATH"

# Create a config file
cp ./config.reference.yaml config.yaml

# Verify installation
microbs help
